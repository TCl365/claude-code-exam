📚 Claude Code - Полное руководство

🤔 Что это

Claude Code = разработчик, который читает весь проект и делает задачи сам.

Ты - архитектор. Claude - исполнитель.

❓ Claude Code vs Cursor

Cursor - помогает писать код

Claude Code - делает задачу полностью

Fix login bug and run tests

❓ Claude Cowork vs Claude Code

  • Cowork - документы
  • Claude Code - код

⚙️ Установка

npm install -g @anthropic-ai/claude-code

claude

📄 CLAUDE.md

Как создать:

touch CLAUDE.md

Пример:

Backend: Symfony
Frontend: React + TS
Rules: no DB changes

Это память проекта.

📦 Skills

Создание:

mkdir my-skill
cd my-skill
touch SKILL.md

Пример:

Write tests using Jest
Mock DB

⚖️ Разница CLAUDE.md vs Skills

  • CLAUDE.md - правила проекта
  • Skills - как делать задачи

🔌 MCP

Что это: подключение Claude к системам

  • GitHub
  • Jira
  • PostgreSQL
  • Slack

Где искать:

  • modelcontextprotocol.io
  • GitHub search: MCP server

Пример:

Read Jira bugs → fix → update status

⚙️ Как создать MCP

Это API сервер

app.get("/tasks", (req,res)=>{ res.json(tasks) })

🕵️ AI Agents

Claude работает циклом:

  1. Планирует
  2. Пишет код
  3. Запускает тесты
  4. Исправляет

Пример:

Fix all bugs in auth module and run tests

✍️ Prompts

❌ Плохо:

fix bug

✅ Хорошо:

Fix login bug in auth.ts. Users get 401. Run tests after fix.

🚀 Как начать

  1. Установить Claude
  2. Создать CLAUDE.md
  3. Дать задачу
  4. Проверить через git diff
  5. Запустить тесты

🔥 Итог

Claude увеличивает скорость разработки в 2-5 раз.
Но только если есть правила.
